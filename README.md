# ü§ù Contribution Request: Enhance Our Hackathon List!

We invite you to contribute to this resource to keep the list of upcoming hackathons and tech events as **current, accurate, and comprehensive** as possible. Your knowledge and contributions are highly valued!

---

## üìù How to Contribute

You can help improve this repository through both **Pull Requests (PRs)** and **Issues**.

### 1. Submitting New Data or Updates (Pull Requests)

Please submit a **Pull Request (PR)** if you are adding new events, updating dates, or providing the official link for a current/upcoming event (e.g., the 2026 links for Amazon or Flipkart).

* **Format:** Your additions or changes must maintain the existing table structure and formatting.
* **Required Data:** Always try to include the **Event Name**, **Expected Timeline**, the official **Current Link**, and reliable **Past Links (2025/2024)** for historical context.
* **Commit Message:** Use clear and descriptive commit messages (e.g., `FEAT: Add official link for Amazon ML Challenge 2026`).

### 2. Reporting Errors or Suggestions (Issues)

Please **open an Issue** if you encounter:

* **Broken Links:** Any link that returns a 404 error or points to irrelevant content.
* **Inaccurate Data:** Incorrect timelines, statuses, or event names.
* **Suggestions:** Ideas for new entries or ways to better organize the list.

---

## üí° Contribution Guidelines

To ensure the quality and consistency of the list, please adhere to these guidelines:

* **Source:** All submitted links and timelines should be from **official, reliable sources** (e.g., company pages, Unstop, MLH).
* **Consistency:** Maintain the column order and Markdown syntax used in the existing table.
* **Clarity:** Ensure the **Expected Timeline** reflects the most likely month(s) for the event to open registration.
